# Learn_c_and_cpp_function

---


```cpp
/* 宣告 fly 函數的原型 */
void fly(int);

int main()
{
    /* 使用函數 */
    fly(5);

    return 0;
}

/* fly 函數實作 */
void fly(int n)
{
    for(auto i=1; i<=n; i++)
    {
        cout << "fly count " << i << endl;
    }
    
    return;
}
```
